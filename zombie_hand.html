Zombie Hand Feb 2015

Project Description

Created for my 3rd year portfolio module at Staffordshire University

Photographs of my own hand were used to create the base for the texture.

Project Details

  • Inital Base mesh created in 3ds Max
  • High Poly sculpt created in Zbrush used to create Normal map
  • Albedo and Metal/Smoothness map created in Photoshop
  • Rendered in Unity Engine
